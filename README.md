# 準備

1. pythonをインストール。
例えば次のサイトを参照。https://www.python.jp/install/windows/install.html


2. コマンドラインで以下を入力。
```
> python -V  
```
Python 3.8.8のように，バージョンが表示されたら成功。

# 実行
コマンドラインで，以下のファイルが入っているディレクトリに移動。
```
bistro.txt
staub.txt
shuttle.txt
frying.txt
menu.py
```
そのあと，以下を実行。
```
> python menu.py
```

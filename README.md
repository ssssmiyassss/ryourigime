# 準備

1. pythonをインストール。
2. コマンドラインで以下を入力。
```
> python -V  
```
Python 3.8.8のように，バージョンが表示されたら成功。

コマンドが見つかりません等のメッセージが出たら，環境変数を設定してPATHを通す。

# 実行
コマンドラインで，以下のファイルが入っているディレクトリに移動。
```
bistro.txt
staub.txt
test.py
test2.py
```
そのあと，以下を実行。
```
> python test2.py
```
